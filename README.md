# GIS-KIGALI-DIGITAL-TWIN-PROJECT

## Overview
The GIS Kigali Digital Twin Project presents a comprehensive Geographic Information Systems (GIS) analysis of Kigali City, Rwanda aimed at supporting informed urban planning and infrastructure development. The project integrates spatial analysis, infrastructure assessment, and service accessibility evaluation to create a digital twin framework for sustainable city development.

The study applies multi-criteria spatial analysis techniques to:
- Assess accessibility to essential public services
- Evaluate infrastructure coverage
- Identify optimal zones for future development
- Support evidence-based urban planning decisions

## Objectives

The project focuses on three major domains:

### 1. Service Accessibility Analysis
- Health facilities accessibility
- Educational institutions accessibility
- Public transportation accessibility

### 2. Infrastructure Coverage Assessment
- Electricity distribution network analysis
- Infrastructure gap identification

### 3. Development Suitability Analysis
- Multi-criteria weighted overlay analysis
- Priority development zone identification

---

## Methodology

The project employed advanced GIS and spatial analysis techniques, including:
- Proximity analysis
- Kernel density estimation
- Weighted overlay analysis
- Multi-criteria decision analysis (MCDA)
- Network accessibility analysis

### Data Sources
- MININFRA Rwanda datasets
- Open spatial datasets for Kigali City

## Key Findings

### Healthcare Accessibility
- Approximately **87%** of Kigali’s population can access health facilities within a **30-minute walking distance**.

### Education Accessibility
- Around **78%** of residents can reach educational institutions within **20 minutes**.

### Public Transportation Accessibility
- About **65%** of residents live within a **10-minute walking distance** of bus stations.

### Infrastructure Gaps
- **12 critical zones** with inadequate infrastructure coverage were identified.
- These zones cover approximately **28.3 km²** and affect an estimated **91,500 residents**.

### Development Suitability
- **73.4 km²** of land was identified as highly suitable for priority development.
- High-suitability zones are concentrated in central districts with existing infrastructure.

### Infrastructure Priority Zones
- **15.7 km²** were identified as priority intervention zones.
- Approximately **48,200 residents** in these areas lack both electricity and essential services.
